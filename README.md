# vue-emoj

> vue编写的仿微信功能的聊天室，包括表情包与常用语处理
## 预览
https://github.com/lyllovelemon/vue-emoj/blob/master/src/gif/a.gif

## 主要功能
+ 多人聊天
+ 支持文字
+ 支持表情包
+ 支持聊天对象模糊搜索
+ 聊天信息时间展示
+ 聊天信息字数控制

## 快速上手

``` bash
# 安装依赖
npm install

# 本地启动
npm run dev

# 启动生产版本
npm run build

```
