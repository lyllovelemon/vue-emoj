# vue-emoj

> vue编写的仿微信功能的聊天室，包括表情包与常用语处理
## 预览
[![Watch the video](https://github.com/lyllovelemon/vue-emoj/blob/master/src/video/vue-emoj.mp4)](http://youtu.be/vt5fpE0bzSY)
## 快速上手

``` bash
# 安装依赖
npm install

# 本地启动
npm run dev

# 启动生产版本
npm run build

```
